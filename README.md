## 📌 Sobre o Projeto

A **Nexora** é uma plataforma web de gestão de investimentos pessoais. Este projeto apresenta a tela de **metas de investimentos**, onde o usuário pode visualizar o progresso de cada meta financeira, acompanhar valores investidos e metas concluídas.

Projeto desenvolvido como atividade acadêmica na **FIAP**, aplicando conceitos de **HTML semântico**, **CSS responsivo** e **Tailwind CSS**.

## ✨ Funcionalidades

- 📊 Visualização de metas em andamento com barra de progresso
- ✅ Seção de metas concluídas
- ✏️ Botões de editar e excluir por meta
- 📱 Layout responsivo — mobile first
- 🖥️ Navbar adaptada para mobile e desktop
- 🎨 Identidade visual dark theme com paleta laranja
